Installation Guide ​

Prerequisites ​

  • Julia: Version 1.6 or higher

    • Download from julialang.org

    • Verify installation: julia --version

  • Git: For cloning the repository

  • 4GB RAM: Minimum for auction simulations

Installation Steps ​

1. Clone the Repository ​

bash
git clone https://github.com/your-username/julia-auction.git
cd julia-auction

2. Activate Project Environment ​

bash
julia --project=.

In the Julia REPL:

julia
using Pkg
Pkg.instantiate()  # Install all dependencies

3. Verify Installation ​

Test with a basic auction:

julia
# Load the main simulation module
include("src/simulation.jl")

# Create a simple configuration
config = default_config()

# Test that modules load correctly
println("Installation successful!")

4. Run Examples ​

Try the included examples:

bash
julia --project=. examples/usage_examples.jl
julia --project=. examples/augmented_auction_demo.jl

Testing Your Installation ​

Run the test suite to verify everything works:

julia
using Pkg
Pkg.test()

Troubleshooting ​

Common Issues ​

Julia version too old:

  • Update Julia to 1.6+ from julialang.org

Package installation fails:

  • Check internet connection

  • Try: Pkg.resolve() then Pkg.instantiate()

Out of memory during large simulations:

  • Reduce simulation parameters in configuration

  • Increase system RAM or use smaller batch sizes

Getting Help ​

  • Check Troubleshooting Guide

  • Review examples in the examples directory for usage patterns

  • See Development Guide for development setup
