Citation Format ​

When citing the Julia Auction System:

bibtex
@software{julia_auction_system,
  title = {Julia Auction System: Implementation of Augmented Uniform Price Auctions},
  author = {Auction Development Team},
  year = {2024},
  url = {https://github.com/organization/julia-auction},
  version = {1.0.0}
}

For academic papers using this system:

latex
\footnote{All auctions were conducted using the Julia Auction System v1.0.0, 
available at https://github.com/organization/julia-auction}
